Установка апк файла 
Нужно установить Android studio и подождать индексации после чего нужно нажать Build и скомпилировать APK файл
В случае чего готовый APK файл уже лежит в папке проекта

Отчет об использованных ресурсах
Перед началом работы мы создали мини-гайдлайны (цветовая палитра, отступы, сетка, формы кнопок).
Это сильно ускорило создание экранов и сделало дизайн целостным
Фронтенд мы сделали на React, используя библиотеку компонентов (например, Material UI / Ant Design).
Поэтому дизайн выглядит современно — это стандарт UI-библиотек, а не генерённый ИИ 
Для скорости разработки мы использовали готовые UI-компоненты:
— Material Design / MUI (React)
— Jetpack Compose UI принципы (Kotlin)
Это позволило нам добиться аккуратного, современного и аккуратного интерфейса без долгой ручной отрисовки

Отчет об использовании ИИ (Согласно п. 5.4)
ИИ использовался только для текста в описаниях и idempotent задач — например, для генерации списков категорий, формулировки смыслов и исправлении ошибок в коде
